¿Para qué sirve el archivo package.json?

El archivo package.json sirve para guardar información como el nombre del proyecto, su versión, los scripts que se pueden ejecutar, así como también las dependencias utilizadas y sus versiones, como React en el caso de este Proyecto.

¿Qué diferencia notaron entre ejecutar un archivo directamente con Node.js y levantar el servidor de React?

Ejecutar un archivo directamente con Node.js en la terminal se ve que el resultado se muestra en la misma terminal y una sola vez, en cambio levantar el servidor de React, se pueden ver los cambios/resultados de la aplicación en el navegador y al instante, sin tener que reiniciar la aplicación.
